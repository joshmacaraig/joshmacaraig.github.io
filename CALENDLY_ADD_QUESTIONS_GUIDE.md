# How to Add Questions to Your Calendly Event
## Step-by-Step Visual Guide

---

## 🎯 **Where to Add Questions in Calendly**

### **Step 1: Go to Your Event Settings**

1. Log into Calendly: https://calendly.com
2. You'll see your event: **"Free MVP Strategy Audit"**
3. Click on the event name (or click the gear icon ⚙️ next to it)
4. This opens the **Event Editor**

---

### **Step 2: Navigate to "Invitee Questions"**

In the Event Editor, you'll see several tabs on the left side:
- Event details
- **Invitee questions** ← **CLICK THIS ONE**
- Notifications and cancellation policy
- Confirmation page
- Workflows (if you have paid plan)

**Click on "Invitee questions"**

---

### **Step 3: Add Questions**

You'll see a section that says **"Add a question"** or shows existing default questions.

By default, Calendly already asks for:
- ✅ Name (already there)
- ✅ Email (already there)

Now you need to add **4 more questions**.

---

## 📝 **Add Each Question One by One**

### **Question 1: Phone Number (Optional)**

1. Click **"+ Add question"**
2. Select question type: **"Phone number"**
3. Question text: `Best phone number (optional - backup if call drops)`
4. **Uncheck** "Required" (make it optional)
5. Click **"Save"**

---

### **Question 2: MVP Idea (MOST IMPORTANT)**

1. Click **"+ Add question"** again
2. Select question type: **"Multi-line text"** or **"Paragraph text"**
3. Question text: `Tell me about your MVP idea (2-3 sentences)`
4. **Check** "Required" ✅
5. In "Placeholder text" field, add:
   ```
   Example: A marketplace connecting freelance designers with startups. Target launch in 2 months. Need help building the booking system and payment flow.
   ```
6. Click **"Save"**

---

### **Question 3: Timeline**

1. Click **"+ Add question"**
2. Select question type: **"Radio buttons"** (single choice)
3. Question text: `What's your timeline?`
4. **Check** "Required" ✅
5. Add these options (click "+ Add option" for each):
   - `ASAP (within 2 weeks)`
   - `Within 1 month`
   - `2-3 months`
   - `Just exploring for now`
6. Click **"Save"**

---

### **Question 4: Budget Range**

1. Click **"+ Add question"**
2. Select question type: **"Radio buttons"** (single choice)
3. Question text: `What's your budget range?`
4. **Check** "Required" ✅
5. Add these options:
   - `Under $1,500`
   - `$1,500 - $3,000`
   - `$3,000 - $5,000`
   - `$5,000+`
   - `Not sure yet / need guidance`
6. Click **"Save"**

---

## ✅ **Final Question List (Total: 6)**

After you're done, you should see:

1. ✅ **Name** (First & Last) - Required - *Default*
2. ✅ **Email** - Required - *Default*
3. ✅ **Phone number** - Optional - *You added*
4. ✅ **Tell me about your MVP idea** - Required - *You added*
5. ✅ **What's your timeline?** - Required - *You added*
6. ✅ **What's your budget range?** - Required - *You added*

## 🚨 **Important Tips**

### **Question Types in Calendly:**
- **One line text** - Short answers
- **Multi-line text** - Longer answers (use for MVP idea)
- **Radio buttons** - Single choice (use for timeline & budget)
- **Checkboxes** - Multiple choices
- **Phone number** - Phone field with validation
- **Dropdown** - Select from list

### **Make These Required:**
- ✅ Name (default)
- ✅ Email (default)
- ✅ MVP idea
- ✅ Timeline
- ✅ Budget

### **Make This Optional:**
- ⚪ Phone number (don't want to create barrier)

## 💡 **Pro Tips**

### **Order Matters:**
Put questions in this order:
1. Name (default)
2. Email (default)
3. Phone (your addition)
4. MVP idea (your addition)
5. Timeline (your addition)
6. Budget (your addition)

You can **drag and drop** to reorder questions!

### **Why This Order:**
- Start with easy questions (name, email)
- Then optional backup (phone)
- Then the important qualifying questions
- Saves budget question for last (after they're invested)

## ❓ **Troubleshooting**

**Q: I don't see "Add question" button**
A: Make sure you're in "Invitee questions" tab, not "Event details"

**Q: Can I change questions later?**
A: Yes! You can edit anytime without affecting existing bookings

**Q: What if I ask too many questions?**
A: Keep it to 6 max. More = lower conversion rate

**Q: Should phone be required?**
A: No - optional is better. Don't create barriers.

**Q: Can I make budget optional?**
A: You can, but required helps you qualify leads better
